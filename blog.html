Discuss the scope of var, let, and const.

var: This var keyword can be accessed from anywhere cause it works in a functional scope or global scope.

let and const: This let and const keyword can be accessed from only a block cause it works in a block scope. They are only accessible after the point at which they are declared. Additionally, let allows variable reassignment, while const declares a constant variable that cannot be reassigned.

Tell us the use cases of null and undefined.

null:We can assign a null to a variable to indicate that it should have no value or point to nothing.

undefined: It means a variable has been declared but has not been assigned a value. Variables are automatically assigned undefined when declared without an initial value. It is a result of accessing properties or variables that don't exist.

What do you mean by REST API?

REST stands for Representational State Transfer. A REST API (Application Programming Interface) is a type of web service that abides by the principles and constraints of REST architecture. It is a set of rules and conventions for building and interacting with web services over HTTP.

RESTful APIs are widely used in web development to create scalable and interoperable web services. They are commonly used for CRUD (Create, Read, Update, Delete) operation, making them a fundamental technology for building modern web applications.
